# Ali Talal — Portfolio

Personal portfolio website for **Ali Talal**, a Marketing Strategist & Social Media Manager.

Built with **pure HTML, CSS & JavaScript** — no framework, no build step. Deploys instantly as a static site.

## Tech
- **HTML5** — semantic, accessible markup
- **CSS3** — custom properties, grid/flex, responsive, dark theme with emerald accent
- **Vanilla JS** — sticky nav, mobile menu, scroll reveals, animated counters, custom carousel
- **Outfit** font (Google Fonts)
- Original **SVG icons** (no emojis / icon fonts)

## Sections
1. Hero
2. About (Who I am)
3. Services (What I do)
4. Work (carousel of ad creatives)
5. Results (What you get)
6. Contact
7. Footer

## Project structure
```
index.html
css/style.css
js/main.js
assets/            optimized webp images, logo & favicons
site.webmanifest
vercel.json
```

## Run locally
It's a static site — just open `index.html`, or serve the folder:
```bash
npx serve .
```

## Deploy (Vercel)
1. Import the GitHub repo into Vercel.
2. Framework preset: **Other** · Build command: *(none)* · Output dir: `./`
3. Deploy.
